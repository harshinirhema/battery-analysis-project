# Battery Data Analysis Project

## Overview
This project analyzes battery discharge data and predicts capacity using machine learning.

## Features
- Discharge curve visualization
- Capacity calculation
- dQ/dV peak analysis
- Machine learning model (Linear Regression)

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
